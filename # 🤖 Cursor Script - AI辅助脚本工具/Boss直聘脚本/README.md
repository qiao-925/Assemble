# 🚀 Boss直聘脚本 - 求职自动化工具

> 自动化求职脚本工具集合，提升求职效率

## 📚 工具概览

本目录汇集了各种Boss直聘平台的自动化求职脚本，涵盖批量投递、职位筛选、简历管理等功能的自动化工具。每个脚本都经过精心设计和测试，能够显著提升求职工作的效率和质量。

## 🔧 核心脚本

### 🚀 投递脚本
- [boss_zhilian_batch_apply.py](boss_zhilian_batch_apply.py) - 批量投递脚本，适合批量投递多个职位，提高投递效率
- [boss_zhilian_real.py](boss_zhilian_real.py) - 真实投递脚本，适合真实求职场景，包含更多验证和安全措施
- [boss_zhilian_simple.py](boss_zhilian_simple.py) - 简化版投递脚本，适合测试和学习使用，功能相对简单

### 📊 数据管理
- [applied_jobs.json](applied_jobs.json) - 已投递职位记录，避免重复投递，跟踪投递状态
- [boss_config.json](boss_config.json) - 配置文件，包含登录信息、投递策略、筛选条件等配置

### 🚀 启动工具
- [启动脚本.bat](启动脚本.bat) - Windows启动脚本，快速启动各种投递脚本

### 📦 依赖管理
- [requirements.txt](requirements.txt) - Python依赖包列表，确保环境配置的一致性

## 🎯 使用指南

### 🚀 投递脚本使用

#### 环境配置
```bash
# 进入Boss直聘脚本目录
cd Boss直聘脚本/

# 安装依赖
pip install -r requirements.txt
```

#### 脚本选择指南
- **batch_apply.py** - 适合批量投递多个职位
  - 功能：批量投递、职位筛选、自动匹配
  - 适用：有明确求职目标，需要快速投递多个职位
  
- **real.py** - 适合真实求职场景
  - 功能：真实投递、安全验证、投递策略
  - 适用：正式求职，需要谨慎和安全的投递流程
  
- **simple.py** - 适合测试和学习
  - 功能：基础投递、简单配置、学习使用
  - 适用：了解脚本功能，测试投递流程

#### 配置说明
- `boss_config.json` - 主要配置文件
  - 登录信息：用户名、密码、登录方式
  - 投递策略：投递频率、职位筛选条件
  - 简历设置：使用哪个版本的简历
  - 安全设置：投递间隔、验证码处理等

- `applied_jobs.json` - 投递记录管理
  - 已投递职位列表
  - 投递时间和状态
  - 避免重复投递

### 🚀 启动脚本使用

#### Windows用户
```bash
# 双击启动脚本.bat文件
# 或命令行运行
启动脚本.bat
```

#### 手动启动
```bash
# 批量投递
python boss_zhilian_batch_apply.py

# 真实投递
python boss_zhilian_real.py

# 简化投递
python boss_zhilian_simple.py
```
